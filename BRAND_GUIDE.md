# STOCKPIT Brand Guide

## Overzicht
STOCKPIT is een moderne, slimme keukenapp met een minimalistisch design en een focus op gebruiksvriendelijkheid. De brand identiteit is gebaseerd op een donkere emerald groen als primaire kleur, gecombineerd met een schone, witte achtergrond en subtiele glassmorphism effecten.

## Typografie

### Font Familie
- **Primair:** System default (San Francisco op iOS, Roboto op Android)
- **Stijl:** Sans-serif, modern, clean

### Font Gewichten

#### **Extra Bold (800)**
- **Gebruik:**
  - Brand naam "STOCKPIT"
  - Hero titles
  - Sectie headers
  - Belangrijke labels

#### **Bold (700)**
- **Gebruik:**
  - Headers
  - Button tekst
  - Brand labels
  - Sectie titels
  - Belangrijke informatie

#### **Semi-Bold (600)**
- **Gebruik:**
  - Subheaders
  - Labels
  - Links
  - Secundaire headers

#### **Medium (500)**
- **Gebruik:**
  - Secundaire tekst
  - Subtitles
  - Meta informatie

#### **Regular (400)**
- **Gebruik:**
  - Body tekst
  - Beschrijvingen
  - Standaard tekst

#### **Light (300)**
- **Gebruik:**
  - Mission statements
  - Grote hero tekst
  - Elegante subtitles

### Font Groottes

- **42px** - Brand naam (welcome page)
- **32px** - Hero titles
- **28px** - Mission statements
- **20px** - Sectie headers
- **18px** - Brand labels, subtitles
- **17px** - Taglines, button tekst
- **16px** - Body tekst (groot)
- **15px** - Body tekst (standaard)
- **14px** - Labels, meta informatie
- **13px** - Secundaire labels, hints
- **12px** - Eyebrow tekst, kleine labels

### Letter Spacing

- **-1.5px** - Brand naam (groot)
- **-0.5px** - Headers, titles
- **-0.3px** - Subtitles, taglines
- **-0.2px** - Secundaire tekst
- **1px** - Uppercase labels (eyebrow tekst)

## Best Practices

1. **Gebruik altijd STOCKPIT Emerald (`#047857`) als primaire kleur**
2. **Houd consistentie in spacing en border radius**
3. **Gebruik glassmorphism subtiel, niet overdreven**
4. **Zorg voor voldoende contrast voor leesbaarheid**
5. **Gebruik het logo consistent op alle pagina's**
6. **Houd de brand naam altijd in hoofdletters: "STOCKPIT"**
7. **Gebruik wit als primaire achtergrond**
8. **Gebruik subtiele shadows voor diepte**
9. **Houd de UI clean en minimalistisch**
10. **Gebruik de juiste font weights voor hiërarchie**
